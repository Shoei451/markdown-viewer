# 三角関数の基礎

## はじめに

三角関数は、直角三角形の辺の比を表す関数です。数学、物理学、工学など、様々な分野で重要な役割を果たします。

## 基本的な三角関数

### sin (サイン)

正弦（サイン）は、直角三角形において**斜辺に対する対辺の比**です。

```
sin θ = 対辺 / 斜辺
```

### cos (コサイン)

余弦（コサイン）は、直角三角形において**斜辺に対する底辺の比**です。

```
cos θ = 底辺 / 斜辺
```

### tan (タンジェント)

正接（タンジェント）は、直角三角形において**底辺に対する対辺の比**です。

```
tan θ = 対辺 / 底辺 = sin θ / cos θ
```

## 重要な角度の値

| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 0° | 0 | 1 | 0 |
| 30° | 1/2 | √3/2 | 1/√3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |
| 90° | 1 | 0 | 定義なし |

## 三角関数の性質

### 相互関係

三角関数には以下のような重要な関係式があります：

1. **ピタゴラスの定理から**
   ```
   sin²θ + cos²θ = 1
   ```

2. **タンジェントの定義から**
   ```
   tan θ = sin θ / cos θ
   ```

3. **余角の公式**
   ```
   sin(90° - θ) = cos θ
   cos(90° - θ) = sin θ
   ```

### 加法定理

三角関数の加法定理は非常に重要です：

```
sin(α + β) = sin α cos β + cos α sin β
sin(α - β) = sin α cos β - cos α sin β

cos(α + β) = cos α cos β - sin α sin β
cos(α - β) = cos α cos β + sin α sin β
```

## 単位円による理解

三角関数は単位円（半径1の円）を使って視覚的に理解できます：

- 中心から円周上の点までの角度がθのとき
- その点のx座標が **cos θ**
- その点のy座標が **sin θ**

> **Point:** 単位円を使えば、90°を超える角度の三角関数も理解しやすくなります。

## 周期性

三角関数には周期性があります：

- sin θ と cos θ は **360°（2π）ごとに繰り返す**
- tan θ は **180°（π）ごとに繰り返す**

```
sin(θ + 360°) = sin θ
cos(θ + 360°) = cos θ
tan(θ + 180°) = tan θ
```

## 実践問題

### 問題1

θ = 30°のとき、以下を求めよ：
1. sin 30°
2. cos 30°
3. tan 30°

**解答:**
1. sin 30° = 1/2
2. cos 30° = √3/2
3. tan 30° = 1/√3

### 問題2

sin²θ + cos²θ = 1 を使って、sin θ = 3/5 のとき cos θ を求めよ（0° < θ < 90°）。

**解答:**

```
sin²θ + cos²θ = 1
(3/5)² + cos²θ = 1
9/25 + cos²θ = 1
cos²θ = 1 - 9/25 = 16/25
cos θ = 4/5  (θは第1象限なので正)
```
